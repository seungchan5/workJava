window + e : 파일 탐색기
ctrl + space : 이클립스 자동완성
syso + 자동완성 : System.out.println() 출력
ctrl + shift + (+) : 글자크기 변경
ctrl + F11 : 실행
ctrl + / : 주석
ctrl + shift+ w : 열린 에디터창을 모두 닫기
ctrl + space : 사용하고 있는 변수목록
ctrl + shift + f : 코드정렬

src : 소스파일이 위치 하는 곳
      .java
bin : 컴파일된 클래스파일이 위치 하는 곳
      .class

ctrl + d : 소스코드 1줄 삭제
alt + 방향키 : 해당 방향으로 소스코드 1줄 이동
ctrl + alt + 방향키 : 해당 방향으로 소스코드 1줄 복사 이동
