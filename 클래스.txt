
클래스를 정의하는 3가지
필드(속성) / 생성자(초기화) / 메소드(기능) 

1. 필드
필드는 속성을 정의한다.
메소드나 생성자 내부에서 선언하는건 지역변수이다.
변수 선언 형태와 유사하게 생겼지만 변수가 아니다.

2. 생성자
생성자는 new 연산자로 호출된다. (new연산자를 만나 생성자 실행)
생성자의 역할은 객체를 초기화 시켜주는 것이다.
같은 이름으로 생성자를 여러개 만드는 것을 "오버로딩"이라고 한다.

생성자의 특징은 1. 클래스와 동일한 이름을 가지는 것 / 2. 반환타입이 없다는 것 이다.

new -> 객체를 생성하여 (힙영역)메모리에 값을 올려주고 참조할 수 있도록 (스택영역에)주소값을 반환해준다.

Student st = new Student()
new 연산자가 Student생성자를 호출하여 Student()라는 객체를 생성한다. 
Student() 객체를 사용하기 위해서 st라는 변수에 객체를 저장한다.

new연산자로 호출하여 생성한 Student()객체의 매개변수가 비어있으면 컴파일 과정에서 컴파일러가 기본생성자를 찾아보고
기본생성자가 없을 경우 자동으로 기본 생성자를 만들어준다.
